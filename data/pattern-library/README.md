# Personal pattern library data

Generated and maintained by the pattern-library CLI.

```bash
npm run pattern-library:seed      # 100 synthetic records + taxonomy.json
npm run pattern-library:bakeoff     # three-arm holdout comparison
npm run pattern-library:stress      # look-ahead, bias, style ablation
```

Replace seed records with your own point-in-time captures over time. See [src/lib/patternLibrary/ARCHITECTURE.md](../src/lib/patternLibrary/ARCHITECTURE.md).
