# Edge Logo Options

Four logo directions for the Edge web app. All use the brand palette: Deep Dark `#0A0B0E`, Electric Green `#00FF88`, Near White `#E8E9ED`.

Open `preview.html` in a browser to compare side-by-side on dark and light backgrounds.

| Option | Concept | Best for |
|--------|---------|----------|
| **A — Signal E** | Green E + ascending signal bars | Default wordmark; "information edge" |
| **B — Razor Wedge** | Angular E with sharp green wedge | Bold identity; breakout / momentum |
| **C — Apex Corner** | Chart axes + trend line at apex | Research-first, analytical tone |
| **D — Bevel E** | Rounded badge, green middle bar on E | Favicon, dock icon, app tile |
