Edge Logo Pipeline

Full brand kit workflow using LogoLoom + Cursor only. No QuiverAI subscription required.

Path A — SVG marks (recommended) Design icon marks as clean SVG in Cursor, pick from options-v2.html, combine with LogoLoom wordmark.
Path B — Raster explore → redraw Use GenerateImage / Recraft / Midjourney for mood boards, then redraw the winner as minimal SVG (best quality).
Path C — Raster → vector trace Run vtracer locally on a flat PNG, then recolor and simplify in LogoLoom optimize_svg.

Steps

Step 1

Pick or design the icon mark

Icon only — no wordmark in the same file. Use brand tokens: accent #00FF88, strokes #E8E9ED, background #0A0B0E.

Six ready-made marks: phase1-marks-v2/ · Compare at options-v2.html

Step 2

Convert wordmark to paths (LogoLoom)

text_to_path with Space Grotesk Bold → font-independent "Edge" outlines.

Output: phase2-wordmark/edge-wordmark-light.svg

Step 3

Combine mark + wordmark

Lock up icon (left) and wordmark (right) in one SVG. Test at 32px favicon and ~200px header width.

Example: phase3-combined/logo-breakout-wedge-light.svg

Step 4

Optimize and export (LogoLoom)

optimize_svg then export_brand_kit → favicon, PNG sizes, OG image, mono variants, BRAND.md.

Output: phase4-brand-kit/ (24 files)
