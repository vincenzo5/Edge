Edge Logo Walkthrough

End-to-end pipeline: QuiverAI generates the icon mark, LogoLoom converts the wordmark and exports the brand kit. Open this file at /brand/walkthrough/index.html.

Phase 1

Generate icon marks (QuiverAI)

Tool: QuiverAI · Model: Arrow 1.1 · Prompt: apex trend line, no text

Generate 4+ variants per concept. Pick the cleanest mark, recolor to brand tokens (#00FF88 / #E8E9ED). The highlighted card is our selection.

Phase 2

Convert wordmark to paths (LogoLoom)

Tool: LogoLoom · text_to_path · Font: Space Grotesk Bold

Text in SVG breaks without the font installed. LogoLoom converts "Edge" to path outlines so it renders everywhere.

Phase 3

Combine mark + wordmark

Manual composition in SVG · Align icon left, wordmark right

Lock up the selected QuiverAI mark with the LogoLoom wordmark. Test at 16px (favicon) and 200px (header).

Phase 4

Optimize and export brand kit (LogoLoom)

Tool: LogoLoom · optimize_svg + export_brand_kit

One command produces favicon, PNG sizes, OG image, mono variants, and BRAND.md.
