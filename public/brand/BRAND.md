# Edge Brand Guidelines

## Colors

| Role | Name | Hex |
|------|------|-----|
| Background | Deep Dark | `#0A0B0E` |
| Surface | Dark Gray | `#12131A` |
| Border | Subtle Gray | `#1E2030` |
| Accent | Electric Green | `#00FF88` |
| Text Primary | Near White | `#E8E9ED` |
| Text Secondary | Cool Gray | `#8B8FA3` |
| Text Muted | Dim Gray | `#5A5E73` |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Display | Space Grotesk | Bold (700) |
| Body | DM Sans | Regular (400) |

## Logo Usage

- The "E" features the Electric Green accent with signal bars
- Minimum size: 24px height
- Clear space: equal to the height of the "E" on all sides
- Never stretch, rotate, or alter the logo
- Use light variant on dark backgrounds, dark variant on light backgrounds
- Mono variants for single-color contexts (white on dark, black on light)

## Logo Files

| File | Usage |
|------|-------|
| `logo-full-light.svg` | Primary logo on dark backgrounds |
| `logo-full-dark.svg` | Logo on light backgrounds |
| `icon-light.svg` | Icon mark with green accent |
| `icon-mono-white.svg` | Single-color white (dark backgrounds) |
| `icon-mono-black.svg` | Single-color black (light backgrounds) |
