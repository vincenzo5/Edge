# Edge Brand Guidelines

## Concept (locked)

**R2 — Mixed Candles**

- Wordmark: “Edge” in **Space Grotesk Bold (700)** (outlined paths)
- Mark: mixed candlesticks behind **dg** (solid = up, hollow = down)
- No caret / cursor line

## Colors

| Role | Name | Hex |
|------|------|-----|
| Background | Deep Dark | `#0A0B0E` |
| Surface | Dark Gray | `#12131A` |
| Border | Subtle Gray | `#1E2030` |
| Accent | Electric Green | `#00FF88` |
| Text Primary | Near White | `#E8E9ED` |
| Text Secondary | Cool Gray | `#8B8FA3` |
| Light background | Off White | `#F4F5F7` |

## Typography

| Role | Font | Weight |

## Icon / favicon (candle mark)

| File | Usage |
|------|-------|
| `favicon.svg` | **Browser tab** — transparent, square (preferred) |
| `favicon-16.png` / `favicon-32.png` | Tab PNG fallbacks (transparent) |
| `favicon.ico` | Legacy tab path (transparent PNG bytes) |
| `icon-light.svg` | Wide SVG mark (transparent) |
| `icon-dark.svg` | SVG mark on dark plate (app / marketing) |
| `icon-16.png` … `icon-1024.png` | App / PWA / apple-touch (opaque plate) |
| `icon-512.webp` | Web-optimized |

Tab favicons must stay **transparent** (no dark plate) so they blend with browser chrome. Keep the opaque plate only for install / home-screen icons.
